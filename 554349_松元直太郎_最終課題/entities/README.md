# タワーディフェンスゲーム

## 概要
    スクリーンの右側から来る敵を迎撃しましょう。ライフは３つで、3体の敵がスクリーン左に到達するとゲームオーバとなります。できるだけ長く生き残りましょう。

## 実行方法
    まずダウンロードしたファイルを解凍します。解凍したファイルにターミナルでディレクトリ移動し、python3 main.py を実行することで起動することができます。

## 操作方法
    wキーで上方向に移動
    sキーで下方向に移動
    spaceキーで射撃
    マウス左クリックで地雷を設置

## 仕様
    自機
    四角で緑色の物体です。

    敵
    四角で赤色です。

    移動：1コスト
    上下移動のみ行うことができます。

    射撃：1コスト
    敵に当てる事で、敵を撃破することができます。

    地雷設置：2コスト
    六角形の見た目をしています。
    設置時は灰色で、起動後は緑色になります。
    設置から2秒後に起動し、接触した敵を倒した後に消滅します。


## 工夫した点
    自機の操作に対してコストを支払わなければいけない仕様にしました。
    移動、射撃に１コスト、地雷の設置に2コストかかります。
    地雷が強いのでコストを高める、起動までのラグを設けるなどの調整を行いました。
    ゲーム開始のオープニングにて操作方法の説明も行いました。
